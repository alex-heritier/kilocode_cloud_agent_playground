# English to Vietnamese Learning Platform

This is an English to Vietnamese learning platform that leverages Spaced Repetition System (SRS) and n+1 strategies to focus on reading and writing skills. The platform is designed to build a daily study habit by reviewing previously learned words and gradually introducing new vocabulary at the learner's own pace, utilizing proven language learning techniques.

## Key Features
- **SRS Integration**: Reinforces memory through spaced intervals for optimal retention.
- **n+1 Strategy**: Introduces new words incrementally, ensuring learners build upon existing knowledge without overwhelming them.
- **Focus on Reading and Writing**: Exercises emphasize practical language use in context.
- **Daily Habit Building**: Encourages consistent, manageable daily study sessions.
- **Simple and Intuitive UI/UX**: Prioritizes user experience for ease of use and engagement.

The app aims to make language learning accessible, effective, and enjoyable for learners progressing from English to Vietnamese.
